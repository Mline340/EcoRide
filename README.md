# EcoRide
ECF Studi EcoRide
Dans le cadre de ma formation de Graduate Développeur web full stack, j'ai du réalisé une application web de covoiturage. 
EcoRide est une stratup française qui a pour objectif de réduire l'impact environnemental des déplacements. Son ambition est de devenir la première platforme de covoiturage pour les voyageurs soucieux de l'environnement ainsi que pour ceux qui recherchent une solution économique pour leurs déplacements.
